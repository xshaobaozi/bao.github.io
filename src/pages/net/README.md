## NET

### 七层协议
     1. 物理层
     2. 数据链路
     3. 网络层
     4. 传输层
     5. 会话层
     6. 表示层
     7. 应用层

### 四层协议

    1. 数据链路层
    2. 网络层
    3. 传输层
    4. 应用层

#### 数据链路

    物理寻址 比特流转换为逻辑传输路线 

#### 网络层
    
    路由协议 控制子网运行 如 逻辑编址 分组传输 路由选择

#### 传输层

    TCP UDP 

#### 会话层

    SSL，TLS协议

#### 表示层

    加密解密，转换翻译，压缩解压缩

#### 应用层

    HTTP，FTP，SMTP，POP3等
